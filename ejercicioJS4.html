Este proyecto muestra una solución para gestión de inventarios.

Proyecto 2

Una aplicación web de tareas diarias con almacenamiento local.

Incluye funciones para agregar, eliminar y editar tareas, y guarda datos en localStorage.

  • Lenguaje: JavaScript
  • Interfaz: HTML/CSS
  • Compatibilidad: Navegadores modernos
